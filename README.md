This is an initializer for extra-* packages.

```bash
npm init extra
# creates the following files:
# - package.json
# - .npmignore, 
# - .travis.yml
# - build.js
# - bundle.js
# - TODO

npm init extra --keywords "dream,go,false"
# creates package.json given name
```

### reference

```bash
npm init extra [options]
# Options:
# -n, --name
# -v, --version
# -d, --description
# -r, --repository
# -k, --keywords
# -a, --author
# -l, --license
```
